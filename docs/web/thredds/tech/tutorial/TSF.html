Writing a THREDDS Servlet

The THREDDS Servlet Framework (TSF) provides programmatic access to: 1) the configuration mechanism for defining the datasets to be cataloged and served; and 2) an API for accessing the configuraiton information, generating THREDDS catalogs, and responding to servlet requests for catalogs and data access.

Which means the TSF can be used by other servlet developers to add THREDDS catalog generation in their web applications.
