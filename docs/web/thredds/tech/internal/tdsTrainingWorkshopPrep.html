TDS Training Workshop Preparations

General Prep

  • Freeze TDS stable version
  • Review TDS Reference and Tutorial documentation
    • Set aside a full week a month before the workshop to focus only on docs
  • Write up evaluation forms

Pre-workshop Email to Participants

  • Class will start with Java and Tomcat installation and hardening/securing Tomcat
    • If you don't need that section, feel free to arrive later (include link to workshop schedule).
    • If you are interested in that section, consider reading the install and security sections and doing the exercises on your desktop before the class (include links to those sections).
    .
  • We will then have a TDS overview presentation. We assume you are familiar with the concepts of scientific data access including access and subsetting of remote datasets. I.e., why you might want your data to be served by a TDS.
  • If you are having any particular TDS problems or have data you're having trouble serving, come prepared to talk about it (bring some sample data). We will have time set aside for discussing your issues and looking at your data.

Ghost Machine Prep

  • Add link on desktop quick start bar for Firefox and Terminal.
  • Add to Firefox:
    • Install "Live Http Headers" Add-on
    • Add "TDS Docs" folder for bookmarks and add:
      • "TDS Home Page": http://www.unidata.ucar.edu/projects/THREDDS/tech/TDS.html
      • "TDS Administrator Tutorial": http://www.unidata.ucar.edu/projects/THREDDS/tech/tds4.2/tutorial/schedule.html
      • "TDS Reference": http://www.unidata.ucar.edu/projects/THREDDS/tech/tds4.2/reference/
    • Open the the "TDS Docs" bookmarks in separate tabs and set them as the Firefox "Home Pages"
    • Add "TDS Instances" folder for bookmarks and add:
      • "TDS localhost:8080": http://localhost:8080/thredds/catalog.html
      • "TDS motherlode:8080": http://motherlode.ucar.edu:8080/thredds/catalog.html
      • "TDS motherlode:9080": http://motherlode.ucar.edu:9080/thredds/catalog.html
    • Webstart ToolsUI
      • Add "ToolsUI webstart" bookmark
      • Launch so that it is downloaded and cached.
      • Open the motherlode:8080 catalog in the THREDDS Catalog Chooser tab.
    • Webstart IDV
      • Add "IDV webstart" bookmark
      • Launch so that it is downloaded and cached.
      • Open the motherlode:8080 catalog in the Dashboard - "Data Choosers" tab - "Catalogs" page.
  • Install OpenOffice (at least Impress for PPT)
