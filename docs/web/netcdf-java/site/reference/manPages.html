CDM man pages

Useful applications that can be called from the CDM library, eg:

 java -Xmx1g -classpath netcdfAll-4.3.jar ucar.nc2.NCdumpW Q:/cdmUnitTest/ncml/nc/test_7.nc

ncdump

Dump the contents of the Similar functionality to the ncdump utility program. By default, just the header (ncdump -h) is output. This application works on any CDM file, not just netCDF files.

  java -Xmx1g -classpath netcdfAll-4.3.jar ucar.nc2.NCdumpW filepath [-cdl | -ncml] [-c | -vall] [-v varName1;varName2;..] [-v varName(0:1,:,12)]

where:

  • filepath : pathname of any CDM file.
  • -cdl : show CDL (strict mode)
  • -ncml : show NcML (default)
  • -c : show data for coordinate variables only
  • -vall : show data for all variables
  • -v varName1;varName2;.. : show data for these variables, use variable's full names (including groups if present)
  • -v varName(0:1,:,12) : show data for a section of this variable only, using F90 section specificiation

nccopy

Copy a CDM dataset to a netCDF-3 (default) or netCDF-4 format. Datasets using features from the extended model require netCDF-4.

 java -Xmx1g -classpath netcdfAll-4.3.jar ucar.nc2.dataset.NetcdfDataset -in <fileIn> -out <fileOut> [-isLargeFile] [-netcdf4]

where:

  • filein : the existing file to copy
  • fileout: the new file to create
  • -isLargeFile: for netcdf-3 only, create a large format file.
  • -netcdf4: write to netcdf-4 format. You must have the netcdf-4 C library loaded.

nccompare

Compare 2 CDM files for semantic equivilence..

 java -Xmx1g -classpath netcdfAll-4.3.jar ucar.nc2.util.CompareNetcdf2 file1 file2 [-showEach] [-compareData]

where

  • -showEach : show details of comparing each object
  • -compareData : compare data also
  • file1 : first file to compare
  • file2: second file to compare

FeatureScan

Scans all the files in a directory to see if they are CDM files and can be identified as a particular feature type

   java -Xmx1g -classpath netcdfAll-4.3.jar ucar.nc2.ft.scan.FeatureScan directory [-subdirs]

where

  • directory : scan this directory
  • -subdirs : recurse into subdirectories
