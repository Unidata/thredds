TDS Services

  • Catalog Services [default: enabled for locally served catalogs; disabled for remote catalogs]
  • Data Access Services:
    • OPeNDAP [default: enabled]
    • NetCDF Subset Service: configure, reference [default: disabled]
    • WCS 1.0 Service [default: disabled]
    • WMS 1.3.0 Service [default; disabled]
  • Metadata
    • ncISO: Dataset Metadata Services [default: disabled]
    • OAI Metadata harvesting [default: disabled]
    • also see: Attribute Conventions Dataset Discovery
  • Server Information [default: enabled]
